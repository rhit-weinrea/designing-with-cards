Designing with Cards

### Features
1. Admin mode where designer makes different cards with potential features. They can also add a "price" to the card that can me visible on a toggle
2. Test user can rank (1-number of cards, sort into groups (no limit on number of cards per group), and buy a feature were they can only chose cards that will not total to over the given budget.)
3. The 4 modes are admin mode, sort mode, group mode, and buy a feature mode
4. Admin can make a new "session" with each test user and can save a snapshot of the users results at any time
5. Admin can make a new "product" with different cards
6. Admin can also edit the cards in each product, even over different sessions


### Tech stack
Node.js, React, (some sort of simple database), store snapshots and card names with info locally
